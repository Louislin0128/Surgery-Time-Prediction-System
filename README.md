# 「醫」刻千金．料「術」如神 ╴ 手術時間預測系統

系統介紹
---------------
* 本系統運用資料處理功能將資料集做前置處理，提供各特徵屬性的重要度資訊以協助其選取最為適切的資料特徵，最後再利用資料探勘相關技術創建分類預測模型，並透過術前資料預估每次手術在進行時所耗用的時間。
* 主要目的是輔助醫療院所的行政人員能準確預測每場手術的時間，降低手術排程的失誤機率，減少手術室的資源浪費，達到效益最大化。


系統特色
---------------
* 彈性化的作業流程選擇  
* 多樣性的客製化資料處理功能  
* 簡單易懂的系統介面設計  
* 支援多種分類預測演算方法
